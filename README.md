## 安装教程：
1. 下载并安装SMAPI；
2. 找到星露谷的游戏文件目录；
3. 将[FS]Wardrobe of Octopath Traveler 及[FS]Wardrobe of Octopath Traveler 2（可以携带版本号后缀）两个文件夹移入Mods文件夹；
4. 通过SMAPI启动游戏。
